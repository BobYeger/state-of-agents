# Building a CLI for all of Cloudflare

## Summary

- Describes Cloudflare's work to make its platform available through consistent CLI commands, SDKs, config, docs, schemas, bindings, Terraform, and Agent Skills.
- Important because it frames agents as a primary customer of APIs and argues that CLI design is context engineering for agents.
- Useful complement to Code Mode: a CLI gives agents a progressive, executable surface when shell access is available.
