# Trajectory-Informed Memory Generation

## Summary

- Extracts actionable learnings from agent execution trajectories: successful strategies, recovery tips, and optimization tips.
- Important because it treats memory as structured experience derived from decisions, failures, recoveries, and inefficiencies rather than generic facts.
- Strong complement to ReasoningBank, skill libraries, and dreaming-style memory consolidation.
