# Cursor 3 Agents Window

## Summary

- Cursor changelog introducing the Agents Window for running many agents across repos and environments.
- Important because it treats the IDE as an agent command center across local sessions, worktrees, cloud, and remote SSH.
- Adds `/worktree` and `/best-of-n` style isolated parallel runs where outcomes can be compared.
