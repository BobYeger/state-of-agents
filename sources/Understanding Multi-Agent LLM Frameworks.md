# Understanding Multi-Agent LLM Frameworks: A Unified Benchmark and Experimental Analysis

## Summary

- Introduces MAFBench, a unified benchmark for comparing multi-agent LLM frameworks.
- Treats framework architecture as an experimental variable, not neutral plumbing.
- Reports large differences in latency, planning accuracy, and coordination success caused by framework choice.
