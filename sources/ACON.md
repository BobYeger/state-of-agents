# ACON

## Summary

- Agent-specific context compression framework for observations and interaction histories.
- Optimizes natural-language compression guidelines using failure cases where full context succeeds but compressed context fails.
- Important because it treats compression as part of long-horizon agent optimization rather than a generic prompt-shortening trick.
