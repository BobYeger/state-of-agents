# On Three-Layer Architectures

## Summary

- Classic robotics architecture note separating control, sequencing/execution, and deliberation/planning.
- Useful historical root for the modern planner-executor-verifier and supervisor-worker patterns in LLM agents.
- The enduring design question is how much control should live in reactive execution versus a slower deliberative layer.
