# ChatDev

## Summary

- Foundational software-agent-team paper that organizes LLM agents into social roles for design, coding, testing, and documentation.
- Important as a prelude to later agent-team work because it treats communication protocol and phase structure as the coordination mechanism.
- Useful contrast to [[sources/MetaGPT]], which pushes the same software-company idea toward standardized operating procedures and structured outputs.
