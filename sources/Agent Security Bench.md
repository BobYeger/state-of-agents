# Agent Security Bench (ASB): Formalizing and Benchmarking Attacks and Defenses in LLM-based Agents

## Summary

- Benchmark for attacks and defenses in LLM-based agents across multiple scenarios and tools.
- Covers prompt injection, memory poisoning, backdoor-style attacks, mixed attacks, and defenses.
- Core benchmark for agent security rather than generic LLM safety.
