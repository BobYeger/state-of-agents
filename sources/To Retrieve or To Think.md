# To Retrieve or To Think?

## Summary

- Proposes an agentic context-evolution loop that decides when to retrieve new evidence and when to reason with existing context.
- Useful for avoiding indiscriminate retrieval that saturates context with irrelevant noise.
- Connects context management with orchestration because a controller decides how working memory evolves.
