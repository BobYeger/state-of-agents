# HiPER

## Summary

- Hierarchical reinforcement-learning method for LLM agents with explicit high-level planning and low-level execution.
- Important for learned plan-execute agents because it separates slow strategic planning from fast local action.
- Connects deliberative control to credit assignment and agent training.
