# Agent Skills Specification and Repository

## Summary

- Open SKILL.md-format specification for portable agent skills.
- Defines skills as folders of instructions, scripts, and resources that agents can discover, load, and use through progressive disclosure.
- Kept as the canonical standard node so platform-specific docs can connect here instead of duplicating the definition.
