# From Lazy Agents to Deliberation

## Summary

- Studies multi-agent reasoning systems where agents can collapse into passive or low-contribution behavior.
- Adds deliberation and incentive mechanisms so agents restart, critique, or influence reasoning rather than merely echoing.
- Useful for understanding why multi-agent systems need explicit contribution and deliberation policies, not only more participants.
