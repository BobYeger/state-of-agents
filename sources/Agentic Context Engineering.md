# Agentic Context Engineering

## Summary

- Treats context as an evolving playbook that accumulates and organizes strategies through generation, reflection, and curation.
- Important because it moves context engineering from static prompt design to an adaptive, self-improving artifact.
- Connects Karpathy-style LLM-maintained knowledge bases with practical agent memory, skills, and context repositories.
