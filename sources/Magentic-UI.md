# Magentic-UI

## Summary

- Human-in-the-loop successor to the Magentic-One style of multi-agent system.
- Adds co-planning, co-tasking, multi-tasking, action guards, long-term memory, web browsing, code execution, file manipulation, and MCP extensibility.
- Important because it treats human oversight and intervention as part of the architecture rather than a fallback after autonomy fails.
