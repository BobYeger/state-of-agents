# SkillsBench: Benchmarking How Well Agent Skills Work Across Diverse Tasks

## Summary

- Benchmark for measuring whether curated or self-generated skills improve agent performance across diverse domains.
- Important corrective: skills can help substantially, but effects vary and self-generated skills are not automatically useful.
- Connects Agent Skills to eval methodology and harness-level measurement.
