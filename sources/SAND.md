# SAND

## Summary

- Trains agents to deliberate over candidate actions using execution-guided feedback.
- Bridges prompting-style deliberation and learned agent behavior: deliberation traces become training data.
- Good anchor for the claim that deliberation can be learned and optimized, not only hand-written into prompts.
