# llm-wiki

## Summary

- Proposes a practical pattern for LLM-assisted research: collect raw source material, then maintain a generated wiki that summarizes, links, and reorganizes the material.
- Separates raw input from synthesized knowledge so the generated layer can be repeatedly regenerated or corrected.
- Fits this vault's operating model: keep immutable source notes and artifacts, then build cross-linked concept maps and claim pages.

## Claims

- A knowledge base becomes more valuable when the model maintains the connective tissue, not just summaries.
- The useful artifact is a wiki-like structure that can be incrementally improved as sources accumulate.
