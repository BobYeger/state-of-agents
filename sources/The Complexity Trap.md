# The Complexity Trap

## Summary

- Compares raw context, observation masking, LLM summarization, and a hybrid strategy for software-engineering agents.
- Finds that simple observation masking can substantially reduce cost while matching or slightly exceeding LLM summarization in the studied settings.
- Important corrective against assuming more complex summarization is automatically better.
