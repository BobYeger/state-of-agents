# AgentDropout: Dynamic Agent Elimination for Token-Efficient and High-Performance LLM-Based Multi-Agent Collaboration

## Summary

- Dynamically removes low-value agents and communication to improve token efficiency.
- Direct evidence against assuming all agents should participate at all times.
- Important for runtime pruning, budget control, and robustness.
