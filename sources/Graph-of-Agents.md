# Graph-of-Agents

## Summary

- Graph-based framework for multi-agent LLM collaboration over a pool of heterogeneous models.
- Selects relevant agents using model-card information, constructs directed communication edges, performs forward and reverse message passing, and pools outputs into a final answer.
- Important frontier source for graph-message-passing and agent-selection architectures: the system can outperform baselines while using fewer selected agents than the full pool.
