# Croto

## Summary

- Cross-team orchestration paper that coordinates multiple agent teams rather than relying on one fixed development chain.
- Important because it moves the team pattern from "one software company" to multiple teams exploring alternative paths and exchanging insights.
- Useful bridge between software-agent teams, topology search, and search-over-workflows approaches.
