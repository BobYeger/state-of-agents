# How Well Do Agentic Skills Work in the Wild: Benchmarking LLM Skill Usage in Realistic Settings

## Summary

- Studies skill usage under realistic conditions where agents must retrieve and select from a large real-world skill collection.
- Important because it shows skill benefits degrade as the setting becomes less hand-curated.
- Pairs well with SkillsBench and skill-retrieval/consolidation work.
