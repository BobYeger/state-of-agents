# ContextBench

## Summary

- Process-oriented benchmark for context retrieval in coding agents.
- Adds human-annotated gold contexts to issue-resolution tasks so agents can be evaluated on recall, precision, and efficiency before final task success.
- Important because it makes context retrieval observable rather than hidden inside pass/fail outcomes.
