# Architecting Resilient LLM Agents

## Summary

- Architecture guide for secure plan-then-execute implementations.
- Useful because it treats planner, executor, optional verifier, replanning, human review, and framework blueprints as production design choices.
- Fits the safety side of deliberative control: separating planning from action creates control points, but also privileged components that need protection.
