# The Ralph Playbook

## Summary

- Repository/playbook describing the Ralph Wiggum coding-agent loop as a practical development method.
- Important as a harness pattern: specs and implementation plans live in files, agent runs are short and repeated, tests provide backpressure, and each loop ends with durable repository state.
- Added as a pattern source only; the Claude Code Ralph plugin is intentionally not included.
