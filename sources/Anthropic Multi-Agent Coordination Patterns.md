# Anthropic Multi-Agent Coordination Patterns

## Summary

- Architecture taxonomy for generator-verifier, orchestrator-subagent, agent teams, message bus, and shared-state patterns.
- Useful because it separates agent teams from one-shot subagent delegation and clarifies when persistent workers are worth the coordination cost.
- Strong anchor for deciding whether a system should be a pipeline, a team, or a shared-state collaboration.
