# Reinforcement Learning for Self-Improving Agent with Skill Library

## Summary

- Introduces SAGE, an RL framework for self-improving agents with accumulated skill libraries.
- Important because it moves from static skills to skill generation, validation, and reuse across related tasks.
- Connects procedural memory to reinforcement learning and deployment-time self-improvement.
