## Summary

- Documents an opinionated Cloudflare Workers base class for the chat-agent lifecycle: loop, persistence, streaming, tool execution, client tools, stream resumption, and extensions.
- Important because it makes durable event streams, subagent mode, and extension points part of the runtime surface.
- This is a narrow runtime primitive, not generic Cloudflare product coverage.
