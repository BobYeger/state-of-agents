# Why Do Multi-Agent LLM Systems Fail?

## Summary

- Systematic failure analysis across multi-agent LLM systems.
- Introduces MAST-Data with annotated traces and a failure taxonomy.
- Groups failures into specification/system design, inter-agent misalignment, and verification/termination problems.
