# TheAgentCompany: Benchmarking LLM Agents on Consequential Real World Tasks

## Summary

- Benchmark simulating work in a software company with websites, internal data, code, and coworker communication.
- Good anchor for consequential digital-worker tasks and long-horizon professional automation.
- Shows current agents handle simpler tasks but remain weak on harder long-horizon workflows.
