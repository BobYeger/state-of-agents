# Multi-Agent Collaboration via Evolving Orchestration

## Summary

- Proposes a puppeteer-style orchestrator that dynamically directs agents as task state changes.
- Bridge between fixed workflows and adaptive orchestration policies.
- Important for the question of when centralized orchestration should remain dynamic rather than scripted.
