# New in Deep Agents v0.6

## Summary

- Introduces a cluster of runtime primitives: code interpreter, harness profiles, typed streaming, delta channels, and ContextHubBackend.
- Important because it names model-specific harness tuning, programmatic tool calling, checkpoint scaling, and event-stream projections as first-class production concerns.
- Strong complement to Anthropic and OpenAI harness material because it exposes equivalent concepts in an open framework stack.
