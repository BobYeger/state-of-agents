# Memory for Autonomous LLM Agents

## Summary

- Survey of memory mechanisms for autonomous LLM agents, formalizing memory as a write-manage-read loop.
- Useful bridge between context compression, retrieval stores, reflection, hierarchical virtual context, and learned memory policies.
- Important because it separates memory as operational substrate from simple transcript recall.
