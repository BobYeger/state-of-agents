# Devin Manages Devins

## Summary

- Devin release-note entry where a main Devin session delegates to a team of managed Devins working in parallel.
- Important product architecture example: each managed Devin is a full Devin in an isolated VM, while the main session scopes work, monitors progress, resolves conflicts, and compiles results.
- Useful as a coding-agent team control-plane example, not as generic Devin marketing.
