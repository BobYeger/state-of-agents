# ChatGPT Agent

ChatGPT agent is OpenAI's unified agent product combining browsing, research, tool use, computer interaction, and artifact creation.

It matters because it merges previously separate agent products and capabilities: browser/computer use from Operator, persistent research behavior from deep research, and user-facing artifact workflows. It is therefore a product-level case study in how agent capabilities get packaged, gated, and safety-reviewed.

## Design Pattern

- Combine research, action, and artifact generation in one user-facing loop.
- Use system-card-style safety reporting for autonomy, tool access, and sensitive actions.
- Gate risky operations through confirmation, policy, and task-specific restrictions.
