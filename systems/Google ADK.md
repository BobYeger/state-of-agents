# Google ADK

Google's Agent Development Kit is a framework for building agent and multi-agent applications, with explicit support for orchestration patterns and integration with Google's A2A direction.

It matters because Google is pairing framework design with interoperability design: ADK is about building agent applications, while A2A is about letting independently built agents discover, communicate, and coordinate.

## Design Pattern

- Model multi-agent systems as composable application workflows.
- Connect orchestration patterns to protocol-level interoperability.
- Keep routing, delegation, and human-in-the-loop patterns visible to the developer.
