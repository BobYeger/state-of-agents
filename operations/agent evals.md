# agent evals

Agent evals measure agent behavior across multi-turn tool use, state changes, autonomy, reliability, safety, and task completion. This operation node keeps evaluation sources connected without duplicating benchmark notes.

## Improvement Claim

Evals improve agent systems when they expose the behavior that matters in deployment: tool use, recovery, cost, latency, safety, state changes, and multi-turn failure modes.
