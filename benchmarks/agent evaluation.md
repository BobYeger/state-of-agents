# Agent Evaluation

Agent evaluation measures behavior across multi-step interaction with tools, environments, users, memory, and changing state.

## Evaluation Axes

- Task success
- Tool-use reliability
- Long-horizon persistence
- Cost and latency
- Safety and policy adherence
- Robustness to prompt injection and tool failure
- Recovery from mistakes
