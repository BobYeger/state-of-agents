# Agent Skills Map

Agent Skills are the reusable procedural-knowledge layer for agents: folders or packages of instructions, scripts, references, and assets that an agent can discover and load only when relevant.

This map keeps standards, platform docs, repositories, and papers together without duplicating the core concept note.

## Role In System Improvement

Skills are one answer to the "what makes agents better?" question: they package reusable procedures so agents do not have to rediscover them every run. The hard part is not only authoring skills, but retrieving the right skill, evaluating whether it helps, and preventing stale or unsafe procedures from spreading.

Related synthesis: [[claims/Claim - Agent memory and skills create compounding improvement loops]] and [[maps/What Makes Agent Systems Better]].
