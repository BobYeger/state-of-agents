# Context Retrieval

Context retrieval is the process by which an agent finds the files, snippets, memories, tool outputs, or documents that should enter the active context window.

For coding agents, final task success is not enough to measure this capability. A good agent must retrieve high-recall context without flooding itself with irrelevant evidence.
