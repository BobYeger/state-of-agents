# Reasoning Memory

Reasoning memory stores reusable strategies, recovery patterns, decision attributions, and task-specific lessons rather than only facts or preferences.

It is closer to an experience library than a transcript archive. The strongest versions preserve provenance, distinguish successful from failed trajectories, and retrieve guidance only when the current task matches.
