# Outcomes and Rubric Graders

Outcomes are explicit definitions of done for agent work. A rubric grader checks an artifact or trajectory in a separate context and returns concrete feedback for repair.

This pattern matters because a generator is often too lenient about its own output. A separate evaluator can be tuned to be skeptical, can run after artifacts exist, and can provide a stable acceptance bar across retries.
